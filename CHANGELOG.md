# 更新日志

## [1.0.0] - 2024-12

### 新增功能
- ✨ 智能行程规划：支持语音和文字输入，AI自动生成旅行计划
- ✨ 费用预算管理：实时记录和追踪旅行开销
- ✨ 用户认证系统：完整的注册登录功能
- ✨ 地图可视化：基于高德地图的路线展示
- ✨ 语音识别：支持浏览器Web Speech API
- ✨ 云端同步：基于Supabase的数据存储和同步

### 技术实现
- 前端：React 18 + TypeScript + Vite
- 后端：Node.js + Express + TypeScript
- 数据库：Supabase (PostgreSQL)
- AI服务：阿里云百炼平台
- 地图服务：高德地图API
